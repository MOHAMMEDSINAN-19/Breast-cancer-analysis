# Breast-cancer-analysis
A Machine Learning project for breast cancer analysis and classification using the Wisconsin Breast Cancer dataset with data preprocessing, visualization, and multiple classification algorithms.

# Breast Cancer Analysis using Machine Learning

## Project Overview

This project focuses on analyzing and classifying breast cancer data using Machine Learning techniques. The objective is to predict whether a tumor is **Benign** or **Malignant** based on various medical features. The project includes data preprocessing, exploratory data analysis, model training, evaluation, and prediction.

---

## Problem Statement

Early detection of breast cancer can significantly improve treatment outcomes. This project develops a machine learning model that classifies breast cancer tumors accurately using the Breast Cancer Wisconsin dataset.

---

## Objectives

- Load and understand the breast cancer dataset.
- Perform data preprocessing and cleaning.
- Conduct Exploratory Data Analysis (EDA).
- Train machine learning classification models.
- Evaluate model performance using different metrics.
- Compare model accuracy and select the best-performing model.

---

## Dataset

- **Dataset:** Breast Cancer Wisconsin Dataset
- **Target Variable:** Diagnosis
  - Malignant (M)
  - Benign (B)

The dataset contains several medical measurements of breast cell nuclei used for classification.
